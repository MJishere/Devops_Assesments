# AWS Windows EC2 Task

## Objective

Create a Windows EC2 instance, connect via RDP, and retrieve system information.

## Steps Performed

1. Launched Windows Server 2022 EC2 instance
2. Configured security group for RDP access
3. Connected using Remote Desktop
4. Executed `systeminfo` command in CMD

## Output

Screenshots attached showing:

* EC2 instance running
* RDP connection
* System information output

## Tools Used

* AWS EC2
* Windows RDP
